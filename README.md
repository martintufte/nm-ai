# Norwegian Championship in AI (NM i AI 2026)

Competition: March 19 (18:00 CET) - March 22 (15:00 CET)
Platform: https://app.ainm.no

## Tasks

### Task 1: Tripletex - AI Accounting Agent
Build a FastAPI endpoint that receives accounting prompts and executes them via the Tripletex API.
See `nmai/tasks/tripletex/description.md`

### Task 2: Astar Island - Norse World Prediction
Predict terrain probability distributions on a 40x40 grid from limited viewport observations.
See `nmai/tasks/astar_island/description.md`

### Task 3: Norgesgruppen - Grocery Product Detection
Detect and classify grocery products on store shelf images using object detection.
See `nmai/tasks/norgesgruppen/description.md`

## Scoring
Final score = average of normalized scores across all 3 tasks (33.33% each).
Unsubmitted tasks score 0.
